Third-I installation script
===========================

Prerequisites
-------------

 *  Python >= 3.5
 *  pipenv
 *  node >= 8.17.0, npm >= 6.13.4
 *  ssh with GitHub access to the repositories

Installation
------------

This script works on top of the provided StereoPi image.

Run it using ssh:

```
./install.sh root@stereopi.lan
```
